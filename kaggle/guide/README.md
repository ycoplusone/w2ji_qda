<h1>캐글 활용</h1></br>
1. 캐글 회원 가입</br>
2. 직원들은 저자가 경진 대회에 제출한 노트북을 복사하고 실행할 준비</br>
3. 노트북에 작성한 신경망 모델 수정</br>
4. 수정한 신경망 모델의 실행 결과를 경진 대회에 제출하고 평가 접수한다.</br>
